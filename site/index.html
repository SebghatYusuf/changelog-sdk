AI-Powered Changelog SDK for Next.js Apps

Add a public changelog feed, secure admin portal, and AI-assisted release writing in one SDK. Built for fast shipping teams that care about quality updates and clean integration.

Get Started Repository

Why teams use Changelog SDK

A complete changelog system for product updates, release notes, and customer-facing announcements.

Public Feed

Ship a searchable changelog timeline at /changelog with filters and pagination.

Admin Portal

Manage changelog entries at /changelog/admin with secure cookie-based authentication.

AI Enhancement

Convert rough notes into polished release updates using OpenAI, Gemini, or Ollama.

Type-Safe Actions

Use typed server actions and Zod-validated inputs to keep publishing reliable.

MongoDB Backed

Persist updates with Mongoose and a singleton DB connection pattern.

Style Isolation

Uses scoped cl- classes to avoid leaking styles into your host app.

Install

Add the SDK directly from GitHub.

bun add github:SebghatYusuf/changelog-sdk#master
# or
npm install github:SebghatYusuf/changelog-sdk#master

What you get

  • Route-ready changelog manager with public and admin flows
  • AI writing support for release notes and changelog summaries
  • Secure admin login helpers and auth checks
  • Sanitized markdown rendering and typed APIs
