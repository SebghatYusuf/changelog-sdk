Features

Everything you need.
Nothing you don't.

A complete, opinionated changelog system that integrates cleanly into any Next.js app without fighting your existing setup.

Public Feed

Searchable changelog timeline at /changelog with tag filters, pagination, and deep-link entries.

Admin Portal

Manage entries at /changelog/admin with secure cookie-based auth. No separate dashboard to host.

AI Enhancement

Transform rough release notes into polished updates using OpenAI, Gemini, or a local Ollama model.

Type-Safe Actions

Fully typed server actions with Zod validation. No runtime surprises when publishing entries.

MongoDB Backed

Mongoose models and a singleton connection pattern. Works with Atlas free tier.

Style Isolation

Scoped cl- CSS classes. SDK styles never leak into your app.

Admin portal

Write and publish entries in seconds

The built-in admin dashboard gives you a full rich-text editor, tag selector, and version field — all behind secure password-based authentication.

  • Create, edit, and delete changelog entries
  • Publish or keep entries as drafts
  • Tag entries for filtering on the public feed
  • bcrypt-hashed admin password, no accounts to manage
yourapp.com/changelog/admin
AI Enhancement

From rough notes to polished release copy

Paste raw commit messages or bullet points. Let GPT-5, Gemini, or a local Ollama model rewrite them into compelling changelog entries.

  • Supports OpenAI, Google Gemini, and Ollama
  • Rate-limiting built in to prevent accidental API spend
  • Review and edit before publishing
